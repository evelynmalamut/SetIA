Directions: A SET is three cards where each feature, when looked at individually, is either all the same OR all different. Each card contains four features: color (red, purple or green), shape (oval, squiggle or diamond), number (one, two or three) and shading (solid, striped or outlined). If you do not see a set, you can add more cards by clicking the button "add more cards". Once you are ready to play, select the play button
